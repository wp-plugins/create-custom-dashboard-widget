Custom Dashboard Widget is the most easy, simple and friendly plugin for creating custom widget for your WordPress Dashboard.

== Description ==

When developing a WordPress website, you may need to create custom widgets for your dashboard. But if you don't know a bit of coding, it may be a little tough for your. With this plugin, you can create fully custom dashboard widgets without writing a single line of code. With the easy and beautiful user interface, you can create multiple dashboard widgets in minutes.

You can also change color and lots of other stuffs in the paid version.

**Features**<br>
1. dd Unlimited Dashboard Widgets<br>
2. Create Custom Title<br>
3. Full WordPress native editor for creating content area<br>
4. Very easy user interface<br>
5. Beautiful Design<br>
6. Simple to use

== Installation ==

**Way 1:**<br>
1. Download the plugin file<br>
2. Unzip the file into a folder on your hard drive<br>
3. Upload via FTP into your plugins (/wp-content/plugins/) folder<br>
4. Activate the plugin from your dashboard<br>

**Way 2:**<br>
1. From Dashboard > Plugins > Add New > Upload the plugin<br>
2. Activate the plugin<br>
